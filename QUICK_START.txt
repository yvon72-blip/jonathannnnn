🚀 DÉMARRAGE RAPIDE :

   Méthode 1 (Recommandée) :
   ─────────────────────────
   $ cd django_quotes
   $ ./start.sh

   Méthode 2 (Manuelle) :
   ──────────────────────
   $ cd django_quotes
   $ pip install Django
   $ python3 manage.py migrate
   $ python3 manage.py runserver

   Puis ouvrez : http://127.0.0.1:8000/
